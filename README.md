# README - Mod Compacter

This is a Fabric mod skeleton that provides a "Compacter" block which can compact 8 blocks into 1 compacted block using a non-grid GUI.

Important notes:
- The build.gradle.kts uses placeholder versions for Loom/Fabric API mappings. You may need to update plugin and dependency versions to match your environment for Minecraft 1.21.1.
- The code is a minimal, illustrative starting point. You'll likely need to implement correct inventories, networking, screen widgets, and data-driven recipe lookups.

Build:
- ./gradlew build

Run in IDE:
- Import the Gradle project in IntelliJ and run the client configuration provided by Loom.

If you want, I can refine the project further (full GUI, hopper support, configurable recipes) or adjust the Gradle versions to exact working ones for Fabric 1.21.1.
